#### Project clone
```
git clone -b master https://github.com/leftlevel/antd-admin.git
```

#### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn serve
```

#### Compiles and minifies for production
```
yarn build
```

#### Lints and fixes files
```
yarn lint
```
